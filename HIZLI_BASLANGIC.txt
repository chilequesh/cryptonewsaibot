🎯 RAILWAY'E 5 DAKİKADA DEPLOY ETME:

ADIM 1: GitHub Repository Oluştur
─────────────────────────────────
1. https://github.com/new ziyaret et
2. Repository adı: cryptonewsaibot
3. Public seç
4. Create repository

ADIM 2: Dosyaları Yükle
─────────────────────────────────
1. Tüm dosyaları indir
2. Local klasöre koy
3. Terminal aç ve klasöre gir:

   cd /path/to/cryptonewsaibot

ADIM 3: Git Push
─────────────────────────────────
   git init
   git add .
   git commit -m "Crypto News AI Bot v14 - Railway Ready"
   git branch -M main
   git remote add origin https://github.com/KULLANICIADI/cryptonewsaibot.git
   git push -u origin main

ADIM 4: Railway Bağla
─────────────────────────────────
1. https://railway.app ziyaret et
2. GitHub hesabınızla giriş yapın
3. "New Project" → "Deploy from GitHub"
4. cryptonewsaibot repository seçin
5. Railway otomatik deploy başlatır

ADIM 5: Environment Variables
─────────────────────────────────
Railway Dashboard'ta:
Project Settings → Variables

Şu değerleri ekleyin:
- DISCORD_WEBHOOK_URL
- NEWSAPI_KEY
- CLAUDE_API_KEY
- TWITTER_BEARER_TOKEN

✅ DONE! Bot 24/7 çalışacak!
